# 产品范围

## 核心价值

帮助用户建立并维持**戒色 / 减少成瘾内容消费**的习惯：记录、提醒、进度可视化、成就与正向反馈等（具体功能列表在实现阶段用独立需求文档或 Issue 细化）。

## 竞品参照

- **Quittr**：交互节奏、激励机制、订阅模式可作为产品与 UX 的参照，**不得复制文案、美术与品牌**；仅学习品类通用模式。

## 范围（初版）

**包含（示例方向，可迭代调整）**

- 本地或端侧优先的数据（ streak、日志、设置）。
- 无账号或可选账号；若未来加同步，须单独规格与隐私评估。
- 与「戒色」目标一致的教育性内容与健康向文案（避免医学诊断表述）。

**不包含（除非单独立项）**

- 后端业务服务（自建 API、用户体系服务器）。
- 社交_feed、实时聊天等强运营功能（可作为远期 idea，不默认实现）。

## 合规与内容

- 遵守 App Store、Google Play 关于**健康、成瘾、成人内容**的相关政策。
- 应用内文案与引导应避免夸大疗效；付费说明清晰、可取消路径明确。

## 成功指标（占位）

上线后由产品定义：留存、订阅转化、功能使用率等；此处仅保留占位，便于 PRD 对接。
